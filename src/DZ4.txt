Блок 1

🟢 Простая — Инкапсуляция

Создай класс BankAccount.

Требования:

поля owner, balance должны быть private;

конструктор принимает владельца и начальный баланс;

методы:

deposit(double amount)

withdraw(double amount)

getBalance()


нельзя снять больше денег, чем есть на счёте.

🟡 Средняя — Наследование

Создай иерархию:

Animal

От него наследуются:

Dog
Cat
Bird

Требования:

у всех есть поле name;

в Animal есть метод makeSound();

каждый потомок переопределяет его:

Dog → "Гав"

Cat → "Мяу"

Bird → "Чирик"



В main() создай массив животных и заставь каждого "подать голос".

🔴 Сложная — Полиморфизм

Есть класс

Employee

Потомки:

Programmer

Designer

Manager


У каждого должна быть своя реализация метода

calculateSalary()

Например:

программист получает оклад + бонус за проекты;

дизайнер — оклад + бонус за макеты;

менеджер — оклад + процент от продаж.


В main() храни их в одном массиве Employee[].

Пройдись циклом и выведи зарплату каждого, не используя instanceof.

🔥 Сложная — Абстрактный класс

Создай абстрактный класс

Transport

Общие поля:

model

speed


Абстрактные методы:

start()
stop()

Потомки:

Car

Plane

Boat


Каждый должен по-своему реализовать запуск и остановку.

Дополнительно сделай обычный метод

printInfo()

который наследуется всеми.

Создай интерфейсы

Flyable

Затем реализуй классы:

💀 Убийство* — Всё вместе

Разработай мини-систему видеоигры.

Есть абстрактный класс

Character

Поля:

name

hp

level


Методы:

attack() // абстрактный
takeDamage()
heal()
printStats()

Интерфейсы:

Mage
Warrior
Healer

Каждый содержит свои способности.

Создай классы:

Knight

Priest

Wizard

Paladin


Требования:

использовать наследование;

использовать полиморфизм;

использовать инкапсуляцию;

использовать абстрактный класс;

использовать интерфейсы;

никаких instanceof;

в main() создать коллекцию персонажей и смоделировать несколько ходов боя, где каждый действует через общий тип Character.


*последняя задача уже близка к уровню хорошей лабораторной работы в вузе и заставляет применять все перечисленные темы одновременно.
